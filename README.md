## ğŸ§  Pelaksanaan Pancasila dalam Kehidupan Sehari-hari
- *Sila 1 (Ketuhanan)*: Menghormati perbedaan agama, tidak memaksakan keyakinan.  
- *Sila 2 (Kemanusiaan)*: Menolong sesama, menghargai hak asasi manusia.  
- *Sila 3 (Persatuan)*: Menjaga kerukunan, tidak memecah belah bangsa.  
- *Sila 4 (Kerakyatan)*: Musyawarah mufakat, menghargai pendapat orang lain.  
- *Sila 5 (Keadilan Sosial)*: Membantu yang lemah, adil dalam pembagian sumber daya.
